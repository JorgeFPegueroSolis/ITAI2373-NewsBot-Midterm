## Dataset

This project uses the BBC News Classification dataset, a public dataset 
originally published on Kaggle by Bijoy Bose 
https://www.kaggle.com/competitions/learn-ai-bbc/data

The dataset contains 2,225 BBC News articles, each labeled with one of 
5 categories: business, entertainment, politics, sport, or tech.

### File Descriptions

- BBC News Train.csv — training set, 1,490 records, that I am going to use to train and evaluate the model.
- BBC News Test.csv — test set, 736 records, used in the project just as a sample in the NewsBotAgent notebook.
